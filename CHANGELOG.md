# Changelog

All notable changes to the "Developer Focus" theme will be documented in this file.

## [1.0.0] - 2025-11-28

### Added
- Initial release of Developer Focus Dark theme
- Comprehensive syntax highlighting for popular programming languages
- WCAG-compliant color scheme with optimized contrast ratios
- Professional UI styling with blue accent colors
- Semantic token support for enhanced highlighting
- Support for JavaScript, TypeScript, Python, Java, C#, Go, and more
- Accessibility features including color-blind friendly palette
- Clean, minimalistic design optimized for long coding sessions
- Git integration with color-coded status indicators
- Error, warning, and info highlighting
- Markdown support with proper formatting
- Terminal color customization
- Comprehensive documentation and installation guide

### Features
- **Eye-strain reduction**: Carefully balanced dark theme colors
- **Professional appearance**: Clean, modern aesthetic suitable for professional environments
- **Language support**: Optimized highlighting for 20+ programming languages
- **Accessibility**: WCAG-compliant contrast ratios and color-blind friendly design
- **Productivity focus**: Minimal distractions with clear visual hierarchy

### Color Palette
- Background: Deep charcoal (#1e1e1e)
- Foreground: Light gray (#d4d4d4)
- Keywords: Purple (#c678dd)
- Strings: Green (#98c379)
- Numbers: Orange (#d19a66)
- Variables: Warm beige (#e8d5b7)
- Functions: Blue (#61afef)
- Classes: Yellow (#e5c07b)
- Comments: Muted gray (#5c6370)
- Accent: Professional blue (#007acc)
